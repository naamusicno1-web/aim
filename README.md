# Tactical Shooter Game - Complete Android Game

A fully functional **tactical shooter game** with advanced aimbot and ESP systems built for Android.

## 🎮 Features

### Core Gameplay
✅ **Real-time Graphics** - Fast-paced action gameplay
✅ **Wave-based Enemy System** - Increasing difficulty
✅ **Projectile Physics** - Realistic bullet ballistics
✅ **Health & Ammo System** - Resource management
✅ **Score & Progression** - Track kills and performance

### Advanced Features
✅ **Aimbot System** - Auto-targeting with adjustable sensitivity
✅ **ESP/Wallhack** - See enemies through walls with distance
✅ **Predictive Aiming** - Lead shots for moving targets
✅ **Auto-Fire Mode** - Automatic weapon firing
✅ **Real-time HUD** - Health, ammo, kills, wave counter

### Customization
✅ **Aimbot Sensitivity (0.1x - 5.0x)** - Adjust targeting speed
✅ **ESP Range Control (50m - 500m)** - Customize detection radius
✅ **Toggle Features** - Turn aimbot/ESP on/off in-game
✅ **Enemy AI** - Smart enemy behavior and shooting

## 📱 Requirements

- **Android 5.0+** (API 21)
- **Target: Android 13** (API 33)
- **RAM: 2GB minimum**
- **Storage: 100MB**

## 🚀 Build Instructions

```bash
git clone https://github.com/naamusicno1-web/aim.git
cd aim
./gradlew assembleRelease
```

APK will be at: `build/outputs/apk/release/aim-release.apk`

## 🎯 How to Play

- **Touch & Drag** - Move aiming reticle
- **Long Press** - Fire weapon
- **Top Buttons** - Toggle AIMBOT/ESP/AUTO FIRE
- **Survive waves** of increasing difficulty
- **Earn points** for kills

## 📊 Features Explained

### Aimbot
- Auto-locks to nearest enemy
- Adjustable sensitivity
- Predictive aiming

### ESP
- See enemies through walls
- Display distance and health
- Customizable range

### Auto-Fire
- Continuous shooting
- Works with aimbot
